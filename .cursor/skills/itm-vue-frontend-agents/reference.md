# 规则与技能映射参考

## 核心索引

- `rules/README.md`：规则目录与使用时机总览。
- `skills/README.md`：技能目录与规则配对关系。

| 文件 | 关注点 | 触发时机 |

## 决策原则

1. 先一致性后“完美分层”：以当前模块既有风格为准。
2. 小页面不强拆目录；复杂页面再做模块化拆分。
3. 先确定规则约束，再执行技能步骤。
4. 每次输出要说明“读取了哪些规则/技能”和“为何这样实现”。

## 推荐最小读取集

任意 Vue 前端开发任务，至少读取：

- `08-通用约束.md`
- `02-编码规范.md`
- `03-项目结构.md`

然后按任务类型再追加页面/API/store/样式/测试相关文件。
