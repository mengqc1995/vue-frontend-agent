# 主题与样式变量

## 使用时机

编写或修改组件样式、需要适配主题时使用。配合规则：09-样式规范。

## 步骤概览

1. **优先使用项目主题与变量**
   - 项目有 `src/theme/` 目录（scss、变量、mixins），优先引用已有变量。
   - 若使用 Element Plus，使用其提供的 CSS 变量（如项目已配置主题变量）。

2. **禁止硬编码颜色与尺寸**
   - 主色、文本色、背景色、边框色等不写死色值，使用变量或 theme 中的定义。
   - 确保浅色/暗色主题切换时表现一致。

3. **类名与 BEM**
   - 样式类名必须符合 BEM；禁止内联样式；动态样式用 CSS 变量或 BEM 修饰符 + class 绑定。

## 产出

- 样式写在 `<style lang="scss" scoped>` 中，使用主题/变量与 BEM 类名，无硬编码颜色。
