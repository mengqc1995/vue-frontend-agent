# 项目规范索引

本目录包含前端项目的开发规范，按模块分类组织，便于按需读取。

| 命名、编码风格 | [02-编码规范](./02-编码规范.md) | 编写或审查代码时 |
| 代码放哪个目录 | [03-项目结构](./03-项目结构.md) | 确定代码应放在哪个目录时 |
| 组件创建与拆分 | [04-组件规范](./04-组件规范.md) | 创建或拆分组件时 |
| 接口定义与调用 | [05-API规范](./05-API规范.md) | 新增或修改接口时 |
| 路由与菜单 | [06-路由规范](./06-路由规范.md) | 新增页面或配置路由时 |
| 全局状态 | [07-状态管理](./07-状态管理.md) | 新增或重构状态时 |

| 注释与文档 | [10-文档规范](./10-文档规范.md) | 编写或审查注释时 |
| 测试要求 | [11-测试规范](./11-测试规范.md) | 确认测试要求时 |

## 使用说明

1. 所有规则文件为 `alwaysApply: false`，按需读取对应文件。
2. 具体落地步骤与示例见 `.agents/skills/` 目录下的技能文件。
