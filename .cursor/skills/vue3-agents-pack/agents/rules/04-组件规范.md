# 组件规范

## 组件结构

- 组件放在 `src/components/` 或 `views/<page>/` 下，每组件单独目录。
- 文件：`index.vue`，样式可为同目录下 `index.scss` 或 `scss/index.scss` 等；必须使用 scoped，禁止全局样式污染。
- Props 必须定义清晰接口；动态 class 使用 classnames 或数组/对象绑定。

## 组件层级

- **页面级**：仅单页使用 → `views/<page>/components/` 或同页面目录下。
- **通用**：跨页面复用 → `src/components/`。
- 单文件建议不超过 400 行，超过时考虑拆分子组件。

## 样式与命名

- 类名遵循 BEM，禁止内联样式（详见 09-样式规范）。
- 组件名与文件名使用 PascalCase（如 `AddCvrDialog.vue`）。
