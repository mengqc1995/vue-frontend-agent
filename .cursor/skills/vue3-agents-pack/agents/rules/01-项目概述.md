# 项目概述

## 项目定位

基于 Vue 3 的管理后台前端项目，面向 USM/ITM 等模式，提供存储管理、任务管理、资源管理、可视化等能力。

## 技术栈

| 领域 | 技术 | 说明 |
|------|------|------|
| UI 框架 | Vue 3 | Composition API、`<script setup>` |
| 类型系统 | TypeScript | 强制使用，禁止 JavaScript |
| 状态管理 | Pinia | 统一使用，禁止 Vuex 等其它方案 |
| 组件库 | Element Plus | 交互 UI 基于此二次封装 |
| 样式方案 | SCSS | 使用 scoped、项目 theme/ 与变量 |
| 路由 | Vue Router 4 | 路由在 router/routes 下按模式拆分 |
| HTTP | Axios | 通过 utils/request 封装 |
| 构建 | Vite | 多模式构建（usm/itm 等） |
